### 4.2 DatabaseHelper 核心代码

```dart
import 'package:sqflite/sqflite.dart';
import 'package:sqflite_common_ffi/sqflite_ffi.dart';
import 'package:path/path.dart';
import 'dart:io';

class DatabaseHelper {
  static final DatabaseHelper _instance = DatabaseHelper._internal();
  static Database? _database;

  factory DatabaseHelper() => _instance;
  DatabaseHelper._internal();

  Future<Database> get database async {
    _database ??= await _initDatabase();
    return _database!;
  }

  Future<Database> _initDatabase() async {
    // Windows 桌面端用 FFI
    if (Platform.isWindows) {
      sqfliteFfiInit();
      databaseFactory = databaseFactoryFfi;
    }

    final path = join(await getDatabasesPath(), 'memo_app.db');

    return await openDatabase(
      path,
      version: 1,
      onCreate: _onCreate,
    );
  }

  Future<void> _onCreate(Database db, int version) async {
    await db.execute(createNotesTable);
    await db.execute(createTodosTable);
    await db.execute(createTodoTypesTable);
    await db.execute(createHabitsTable);
    await db.execute(createHabitTagsTable);
    await db.execute(createCheckinRecordsTable);
    await db.execute(createSettingsTable);
  }
}
```

## 六、各模块开发要点

### 6.1 首页

三个可点击的统计卡片，垂直排列：
- **待办卡片**：总条数、已完成、今日截止、未完成
- **打卡卡片**：今日已打卡/未打卡数量 + 3-4条快捷预览
- **笔记卡片**：上次编辑的笔记标题预览

右下角 FAB 快速添加按钮。

数据查询用 Riverpod `FutureProvider`，自动响应变更。

### 6.2 笔记模块

**列表页**：
- `flutter_staggered_grid_view` 实现瀑布流，移动端2列
- 卡片：可选图片缩略图 + 标题 + 内容摘要 + 时间
- 置顶标记（📌）
- 搜索：标题 + 正文纯文本模糊匹配

**编辑页**：
- `flutter_quill` 富文本编辑器
- 1秒防抖自动保存 + 退出强制保存
- 图片插入：复制到 `<应用目录>/images/`，数据库只存相对路径
- 空笔记丢弃（标题和内容都为空时不保存）
- 标题为空时自动取正文前10字

### 6.3 待办模块

**列表页**：
- 线性列表，每条含复选框 + 标题 + 截止日期 + 优先级
- 完成动画：300ms 斜线置灰 → 400ms 移动到底部
- 展开/收起备注
- 排序：置顶 → 今日截止 → 截止日期 → 优先级 → 编辑时间
- 搜索 + 类型筛选

**创建弹窗**：
- 输入框 + 类型下拉 + 连续添加（回车加入清单）
- 优先级选择（高/中/低）
- 日期时间选择器
- 批量创建

**类型管理**：
- 内置3个：工作、生活、学习
- 最多2个自定义类型（4字以内）

### 6.4 打卡模块

**列表页**：
- 线性列表，每条左右分区
- 左侧：图标 + 名称 + 类型标签 + 累计时长
- 右侧：倒计时/正向计时显示"开始"按钮，不计时显示"完成"按钮，已完成显示"已完成"（置灰）
- 排序：置顶 → 今日未完成 → 累计时长 → 最近打卡

**计时页（正向计时）**：
- 超大字号显示计时时间（核心视觉焦点）
- 今日累计 + 历史累计统计
- "结束打卡"按钮
- 时间计算：当前时间 - start_time，每秒刷新

**计时页（倒计时）**：
- 超大字号显示剩余时间
- 进度条（剩余比例）
- 计划时长显示
- 快捷加时：+5分、+10分、+30分
- 倒计时结束提醒（本地通知 + 音效）
- 超时显示负向计时

**创建弹窗**：
- 输入名称 + 标签选择
- 类型单选：倒计时 / 正向计时 / 不计时
- 倒计时类型显示时长选择（10分/30分/自定义）
- 正向计时/不计时显示提示文字

**每日重置**：
- 应用启动时检测日期变化
- 重置所有 habits 的 today_duration、today_count
- 重置今日完成状态

### 6.5 设置页

**分组列表布局**：

| 分组 | 内容 |
|------|------|
| 数据管理 | 导出数据、导入数据、回收站、**清除所有数据** |
| 个性化 | 主题、字体设置、提醒音效 |
| 安全与隐私 | 应用锁 |
| 其他 | 使用帮助、分享应用、关于 |

**导出数据**：
- 格式选择：CSV / TXT / JSON
- 内容选择：笔记 / 待办 / 打卡 / 全部
- 调用系统文件选择器选保存路径

**导入数据**：
- 系统文件选择器选文件
- 预览条数 + 冲突提示
- 策略：智能合并 / 全部覆盖 / 仅新增
- 去重规则

**清除所有数据**：
- 红色文字入口
- 二次确认弹窗（列出删除内容 + 不可恢复警告）
- 清除：数据库所有表、图片文件、SharedPreferences、通知
- 重置后恢复初始化数据，自动返回首页

**回收站**：
- 按模块分标签页
- 恢复 / 彻底删除 / 清空
- 自动清理：7天后永久删除

## 七、数据清除实现

```dart
class DataClearService {
  final DatabaseHelper _db;
  final SharedPreferences _prefs;

  DataClearService(this._db, this._prefs);

  Future<void> clearAllData() async {
    final db = await _db.database;

    await db.transaction((txn) async {
      await txn.delete('notes');
      await txn.delete('todos');
      await txn.delete('habits');
      await txn.delete('checkin_records');
      await txn.delete('recycle_bin');
      await txn.delete('settings');
    });

    // 重新插入初始化数据
    await db.execute(createTodoTypesTable);
    await db.execute(createHabitTagsTable);

    // 清除本地配置
    await _prefs.clear();

    // 删除图片
    final appDir = await getApplicationDocumentsDirectory();
    final imagesDir = Directory('${appDir.path}/images');
    if (await imagesDir.exists()) {
      await imagesDir.delete(recursive: true);
      await imagesDir.create();
    }

    // 取消所有通知
    await flutterLocalNotificationsPlugin.cancelAll();
  }
}
```

## 九、关键注意事项

### 9.1 确保卸载清除数据

**Android**：在 `AndroidManifest.xml` 中关闭云端备份

```xml
<application
    android:allowBackup="false"
    android:fullBackupOnly="false"
    tools:replace="android:allowBackup">
```

**Windows**：用 MSIX 打包，卸载自动清除所有数据。

### 9.2 图片生命周期

- 插入时复制到应用私有目录，防止原文件被删
- 删除笔记时同步删除对应图片文件
- 避免孤儿文件

### 9.3 计时机制

- 记录时间戳，进入页面时计算，不依赖后台持续运行 Timer
- 切后台不停止，恢复时自动追平
- 应用被杀死：重启时检查 `is_timing == 1`，提示恢复或结算

### 9.4 每日重置

```dart
Future<void> checkDailyReset() async {
  final prefs = await SharedPreferences.getInstance();
  final lastDate = prefs.getString('last_open_date');
  final today = DateTime.now().toIso8601String().substring(0, 10);

  if (lastDate != today) {
    // 重置所有打卡项的今日数据
    await habitRepository.resetTodayData();
    await prefs.setString('last_open_date', today);
  }
}
```

## 十一、无服务端说明

本文档方案**完全不需要以下资源**：

| 不需要 | 说明 |
|--------|------|
| 服务器/VPS | 无后端服务 |
| 域名 | 无 Web 服务 |
| 数据库服务器 | 用 SQLite 本地存储 |
| 对象存储（OSS/S3） | 图片存本地私有目录 |
| 短信服务 | 无登录功能 |
| 微信/QQ 开发者账号 | 无第三方登录 |
| SSL 证书 | 无 Web 服务 |

所有数据完全保存在用户设备本地，导出/导入通过本地文件操作完成。
