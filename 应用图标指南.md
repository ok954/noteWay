# 应用图标获取与添加指南

## 一、图标来源推荐

### 方案 1：AI 生成（最推荐，个人自用最省事）

用 AI 图像生成工具，输入提示词直接生成，再用工具裁剪成方形。

**推荐工具**：
| 工具 | 说明 | 费用 |
|------|------|------|
| **即梦（抖音）** | 中文提示词，效果较好 | 免费额度足够 |
| **可灵（快手）** | 中文提示词，国产 | 免费 |
| **通义万相** | 阿里出品，中文友好 | 免费 |
| **Midjourney** | 效果最佳 | 付费 |
| **Leonardo.ai** | 每日有免费额度 | 免费+付费 |

**提示词参考**（备忘录/笔记类应用）：
```
一个简单的备忘录应用图标，浅色背景，
一个精致的笔记本和一支铅笔，
扁平化设计，简洁现代风格，
柔和的色彩，圆角方形，
没有文字，干净的构图
```

```
A simple memo app icon, light cream background,
a minimal notebook with a small pencil,
flat design, clean modern style,
soft pastel colors, rounded square shape,
no text, clean composition
```

**生成后处理**：
- 用在线工具裁剪成 1:1 正方形（如 remove.bg、稿定设计）
- 确保主体居中，四周留适当边距

### 方案 2：在线图标设计工具（零设计基础）

| 工具 | 网址 | 说明 |
|------|------|------|
| **IconKitchen** | icon.kitchen | 专为 Android/iOS 设计，可调整颜色、形状、图标 |
| **AppIcon.co** | appicon.co | 上传图片自动生成全套尺寸 |
| **Canva** | canva.cn | 有大量模板，拖拽式设计 |
| **Figma** | figma.com | 专业设计工具，社区有很多图标模板 |
| **LogoMakr** | logomakr.com | 简单拖放生成 Logo |

**推荐流程（IconKitchen）**：
1. 打开 https://icon.kitchen
2. 选择背景颜色（如淡黄、浅蓝、米白）
3. 选择图标（搜索 "note"、"memo"、"check" 等）
4. 调整圆角、阴影、渐变
5. 下载 PNG（1024×1024）

## 二、图标规范

### 2.1 源图要求

| 属性 | 要求 |
|------|------|
| **尺寸** | 1024×1024 像素（正方形） |
| **格式** | PNG（透明背景或纯色背景均可） |
| **形状** | 正方形，内容居中，四周留边距 |
| **风格** | 扁平化或轻拟物，简洁为主 |

### 2.2 各平台图标尺寸

Flutter 用工具自动生成，无需手动切：

| 平台 | 尺寸 |
|------|------|
| Android 启动图标 | 多种密度：48/72/96/144/192/512 dp |
| Android 通知图标 | 24/36/48/72/96 dp |
| iOS 图标 | 多种：20/29/40/60/76/83.5/1024 pt |
| Windows 图标 | 16/32/48/256 像素 |

## 三、Flutter 中添加图标

### 3.1 安装 flutter_launcher_icons

```bash
flutter pub add dev:flutter_launcher_icons
```

或在 `pubspec.yaml` 中：

```yaml
dev_dependencies:
  flutter_launcher_icons: ^0.13.1
```

然后运行：
```bash
flutter pub get
```

### 3.2 配置 pubspec.yaml

在 `pubspec.yaml` 末尾添加：

```yaml
flutter_launcher_icons:
  android: true
  ios: false          # 不需要 iOS
  image_path: "assets/icon/app_icon.png"   # 主图标路径
  min_sdk_android: 21
  remove_alpha_ios: true
  
  # Android 自适应图标（可选，效果更好）
  adaptive_icon_background: "#FFFFFF"      # 背景色
  adaptive_icon_foreground: "assets/icon/app_icon_foreground.png"  # 前景图标
  
  # Windows
  windows:
    generate: true
    image_path: "assets/icon/app_icon.png"
    icon_size: 48
```

**目录结构**：
```
assets/
  icon/
    app_icon.png              # 1024×1024 主图标
    app_icon_foreground.png   # 前景图标（如只需要纯色背景可省略）
```

**注意**：`assets/icon/` 目录需要先在项目中创建，然后把图标放进去。

### 3.3 自动生成图标

```bash
flutter pub run flutter_launcher_icons
```

执行后工具会自动：
- 生成 Android 各尺寸图标到 `android/app/src/main/res/mipmap-*/`
- 生成 Windows 图标到 `windows/runner/resources/`
- 自动修改 `AndroidManifest.xml` 中的图标引用

### 3.4 验证效果

```bash
# Android
flutter run

# Windows
flutter run -d windows
```

在设备/模拟器上查看应用图标是否正确显示。

## 四、快速上手流程（推荐）

**如果你不想折腾设计，最快路径**：

1. **打开 IconKitchen**：https://icon.kitchen
2. 选个喜欢的颜色（如浅米色 `#FFF8E7`）
3. 搜索图标关键词 "memo" 或 "notebook"
4. 调整一下圆角大小
5. 下载 1024×1024 PNG
6. 放到项目 `assets/icon/app_icon.png`
7. 按上面步骤配置 `flutter_launcher_icons`
8. 运行 `flutter pub run flutter_launcher_icons`
9. 完成

整个过程 10 分钟搞定。

## 五、常见问题

| 问题 | 解决 |
|------|------|
| 图标生成后还是默认 Flutter 图标 | 检查 `pubspec.yaml` 路径是否正确，重新运行命令 |
| Android 图标边缘被裁切 | 源图四周留出 10%-15% 的边距，不要把内容顶到边缘 |
| Windows 图标模糊 | 确保源图是 1024×1024，不要用小图放大 |
| 图标颜色不对 | Android 自适应图标需要单独设置背景和前景 |
| 想换图标 | 替换源图后重新运行 `flutter pub run flutter_launcher_icons` |
