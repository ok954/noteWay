# 一、产品核心定位
记途是一款聚焦“轻量实用、隐私安全、多端协同”的全场景备忘录工具，基于Flutter跨端框架开发，适配iOS、安卓、Windows多平台，以“拒绝功能过载、回归记录本质”为核心理念，覆盖离线记录、待办事项管理、云端同步、数据安全四大核心需求，兼顾普通用户的日常记录与中小办公群体的高效管理，打造“轻量不简单、稳定更安全”的差异化备忘录产品，填补中低端市场“实用+亲民”的产品空白。

核心目标用户：18-45岁，有日常记录、任务管理需求的普通用户；需要多端同步、数据安全存储的中小办公群体；追求简洁操作、无冗余干扰的学生群体，精准匹配市场38%有隐私安全需求、26%偏好亲民定价的用户群体需求。

# 二、核心功能亮点
产品聚焦核心需求，规避功能过载问题，保留“实用、流畅、安全”的核心体验，同时借鉴成熟产品优势，优化细节功能，具体亮点如下：

## 2.1 基础记录功能：简洁高效，离线可用
+ 极简编辑体验：界面干净无冗余，支持富文本编辑（字体调整、加粗、斜体），操作直观，新手可快速上手，避免复杂导航带来的使用门槛，契合29%用户对“界面简洁、操作流畅”的需求。
+ 离线优先设计：无网络环境下可正常完成笔记的新增、编辑、删除、查询，联网后自动同步至云端，适配户外、旅行等无网/弱网场景，确保记录不中断，参考小黄条备忘录的全场景离线适配优势。
+ 数据导出/导入：支持CSV、TXT两种常用格式，可自由备份笔记数据，也可快速导入其他备忘录内容，解决用户数据迁移痛点，提升产品兼容性。

## 2.2 待办事项管理功能：精准高效，杜绝遗漏
+ 全维度任务设置：支持截止日期、优先级（高/中/低）、重复提醒设置，可标记任务完成状态，实现“记录-执行-归档”全流程闭环，适配办公规划、学习计划等多场景需求。
+ 多元提醒方式：支持桌面弹窗、手机铃声、微信同步提醒三种方式，覆盖办公（电脑端）、移动（手机端）全场景提醒需求，提醒精准度高，有效避免重要事项遗漏，借鉴敬业签的多场景提醒逻辑。
+ 数据统计可视化：首页自动统计待办事项总条数、今日截止条数、已完成条数，直观呈现任务进度，助力用户高效管理时间，贴合办公群体的任务管控需求。

## 2.3 多端同步功能：无缝衔接，信息随行
+ 全平台适配：支持iOS、安卓、Windows多端部署，采用Flutter框架实现一次开发、多端适配，界面一致性高，适配不同终端的界面布局与操作习惯，参考便笺星（NoteStar）的跨端开发优势。
+ 实时同步稳定：多端登录同一账号，笔记、待办事项任务修改实时同步，同步延迟≤10秒，Windows端支持桌面悬浮窗、快捷添加入口，移动端支持手势操作，实现“信息随人动”，打破多设备数据孤岛，契合62%主流应用的多端同步标配趋势。
+ 灵活登录方式：支持手机号登录、微信/QQ快捷登录，无需复杂注册流程，同时保障账号安全，提升用户使用便捷性，参考市场主流应用的生态集成逻辑。

## 2.4 数据安全功能：多重防护，安心存储
+ 双重加密存储：本地数据采用加密算法存储，云端备份支持端到端加密，符合GDPR等全球隐私法规，杜绝数据泄露、篡改风险，贴合38%用户的隐私安全需求，借鉴Notion的合规存储设计。
+ 误删恢复机制：内置时间轴功能，支持笔记、待办事项任务误删恢复，可回溯30天内的操作记录，避免因误操作导致的数据丢失，参考敬业签的数据安全细节设计。
+ 异常容错处理：优化本地存储逻辑，采用SharedPreferences实现JSON数据持久化，避免因数据解析异常导致应用崩溃，提升产品稳定性，借鉴便笺星（NoteStar）的容错设计优势。

# 三、多端适配详情
产品基于Flutter框架开发，实现多端统一体验，兼顾不同终端的使用场景，具体适配详情如下，确保多平台上架合规、体验流畅：

+ iOS端：适配iOS 13.0及以上版本，支持iPhone、iPad设备，适配小屏操作与平板横屏模式，支持iCloud辅助备份，贴合苹果生态用户使用习惯，参考Bear（熊掌记）的苹果设备适配逻辑。
+ 安卓端：适配Android 8.0及以上版本，兼容华为、小米、OPPO、vivo等主流品牌机型，优化低配置设备运行流畅度，体积小巧，占用系统资源少，无卡顿问题。
+ Windows端：适配Windows 10及以上版本，支持桌面悬浮窗、快捷添加入口，可最小化至托盘，支持离线编辑，联网后自动同步，适配办公场景的高效使用需求，参考小黄条备忘录的桌面适配优势。
